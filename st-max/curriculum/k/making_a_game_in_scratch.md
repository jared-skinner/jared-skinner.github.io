# Making a Game in Scratch 

## Goals

* Get a taste of how a computer program works
* Have fun!

## Materials

* Computer 
* Clever Touch
* Scratch

## Lesson Plan

* Talk about what a programming language is
* build out game of taco getting muffins
  * movement
  * bananas
    * teleport when touched
  * score
  * floating muffins
  * difficulty
